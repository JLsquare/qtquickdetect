QtQuickDetect

Une application pour réaliser des détections d'objets sur différents types de supports : images, vidéos, et livestreams (ex. caméra).

Fonctionnalités

  • Détection sur image : Détecter des objets.
  • Détection sur vidéo : Détecter et tracker des objets.
  • Détection sur livestream : Détecter des objets en temps réel.

Téléchargement

Télécharger la dernière version (.zip)

Lancement

Pour pouvoir exécuter l'application, une version de Python doit être installée sur votre système.

Linux

Exécutez le fichier run-linux.sh, si c'est la première fois, choisissez si vous voulez installer CUDA ou non (pour les machines avec un GPU NVIDIA)

Windows

Exécutez le fichier run-windows.bat, si c'est la première fois, choisissez si vous voulez installer CUDA ou non (pour les machines avec un GPU NVIDIA)

Code source

Le code source de l'application est disponible sur GitLab.

Comment utiliser

Pour commencer à utiliser QtQuickDetect, téléchargez la version correspondant à votre système d'exploitation, décompressez l'archive, et exécutez l'application.
